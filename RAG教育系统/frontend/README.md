# Umi + React 示例项目

这是一个简单的 Umi + React 项目示例。

## 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run start
```

## 项目结构

```
├── src/
│   ├── pages/          # 页面组件
│   │   └── index.tsx   # 首页
│   ├── app.tsx         # 应用配置
│   └── global.css      # 全局样式
├── umirc.ts            # Umi 配置
├── tsconfig.json       # TypeScript 配置
└── package.json        # 项目依赖
```

## 技术栈

- Umi 4.6.23
- React 18.3.1
- TypeScript 5.9.3
- Ant Design 5.29.3
```
